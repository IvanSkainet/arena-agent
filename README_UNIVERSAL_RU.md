# 🤖 Arena Local Agent — Универсальное руководство

**Версия:** 1.3.0-unified
**Платформы:** Windows 10/11, Linux (все дистрибутивы), macOS  
**Автор:** Сообщество Arena Agent

---

## 1. Что это такое?

**Arena Local Agent** — это ваш личный, полностью локальный ИИ-ассистент. Он запускается на вашем компьютере и даёт любым ИИ-моделям (ChatGPT, Claude, Gemini, Arena.ai) безопасный доступ к вашей системе: файлам, терминалу, браузеру, рабочему столу.

**Что умеет агент:**
- Выполнять команды в терминале (PowerShell / bash).
- Управлять файлами (читать, писать, искать).
- Делать скриншоты экрана и управлять браузером.
- Работать в фоне через очередь задач (`agentctl do`).
- Устанавливать расширения MCP (например, управление рабочим столом).
- Показывать красивый веб-дашборд по адресу `http://127.0.0.1:8765/gui`.

## 2. Требования (сделать один раз)

### ✅ Windows 10 или 11 (64-бит)
Агент работает на «чистой» Windows без WSL.

### ✅ Python 3.10 или новее
1. Откройте браузер и зайдите на официальный сайт:  
   👉 **https://www.python.org/downloads/**
2. Нажмите большую жёлтую кнопку **"Download Python 3.xx.x"**.
3. Запустите скачанный файл `python-3.xx.x-amd64.exe`.
4. **ВАЖНО!** На первом экране установщика поставьте **галочку**:
   > ☑ **Add python.exe to PATH**  
   > (внизу окна, слева)
5. Нажмите **Install Now** и дождитесь окончания.
6. Чтобы проверить, откройте PowerShell (`Win + X`, выберите *Terminal* или *Windows PowerShell*) и введите:
   ```powershell
   python --version
   ```
   Должно появиться что-то вроде `Python 3.12.4`. Если пишет "не найдено" — перезагрузите компьютер.

### ✅ Tailscale (для безопасного доступа из интернета)
Tailscale создаёт зашифрованный туннель между вашим ПК и ИИ-чатом. Никаких открытых портов и IP-адресов.

1. Зайдите на сайт: 👉 **https://tailscale.com/download/windows**
2. Скачайте и установите `Tailscale-setup.exe`.
3. После установки в системном трее (рядом с часами) появится значок Tailscale (чёрная иконка с хвостиком).
4. **Зарегистрируйтесь:**
   - Нажмите на иконку → **Log in…**
   - Откроется браузер. Выберите способ входа (Google, Microsoft, GitHub или email).
   - Если у вас ещё нет аккаунта — просто войдите через Google, аккаунт создастся автоматически.
   - Разрешите подключение устройства (`Connect`).
5. В системном трее иконка станет **зелёной** — значит, вы в сети.

> 💡 **Если Tailscale заблокирован вашим провайдером** (не получается залогиниться), используйте Zapret, VPN или напишите нам — мы подскажем альтернативу (ngrok / Localtonet).

## 3. Установка Агента (2 минуты)

1. **Скачайте архив** `arena_agent_release.zip` (вам пришлют его в Discord / Telegram / другое место).
2. **Распакуйте** архив прямо в вашу **пользовательскую папку**:
   - Нажмите `Win + E` (откроется Проводник).
   - Слева выберите папку с вашим именем пользователя (например, `C:\Users\Вася`).
   - Перетащите туда содержимое архива. Должно получиться две папки:
     ```
     C:\Users\Вася\arena-agent
     C:\Users\Вася\arena-local-bridge
     ```
3. **Запустите установщик:**
   - Откройте папку `arena-agent`.
   - Найдите файл `install.bat`.
   - **Дважды кликните** по нему.
   - Если Windows спросит "Разрешить этому приложению вносить изменения?" — нажмите **Да**.
4. **Ждите.** Чёрное окно покажет прогресс:
   - Проверка Python.
   - Автоустановка Node.js (если не стоит).
   - Авто-регенерация токена безопасности (каждый раз новый, сохраняется в token.txt).
   - Проверка версий опциональных компонентов и авто-обновление.
   - Регистрация 1 фоновой службы (Unified Bridge на порту 8765).
5. Когда окно напишет **"INSTALLATION COMPLETE"**, нажмите любую клавишу.

> ⚠️ **Не закрывайте окно раньше времени!** Если закроете на середине — просто запустите `install.bat` ещё раз, он доведёт до конца.

## 4. Как получить токен безопасности

Токен — это ваш "пароль", который вы даёте ИИ, чтобы он мог управлять агентом. Токен уникальный для каждого компьютера.

> **Авто-регенерация:** При каждом запуске `install.bat` токен автоматически генерируется заново. Не нужно делать это вручную!

1. Откройте PowerShell или Командную строку (`Win + R` → `cmd` → Enter).
2. Введите команду:
   ```cmd
   type %USERPROFILE%\arena-local-bridge\token.txt
   ```
3. Выделите и **скопируйте** длинную строку букв и цифр (например, `dB9x...A2q`).
4. Сохраните её временно в блокнот — скоро понадобится.

> 💡 Если нужно сменить токен без переустановки, запустите `regenerate_token.bat` в папке `arena-local-bridge`.

## 5. Как подключить ИИ к вашему компьютеру

### Шаг А — включить "туннель" через Tailscale

Чтобы ИИ из интернета мог достучаться до вашего ПК, нужно активировать Tailscale Funnel.

1. Убедитесь, что Tailscale запущён (иконка в трее зелёная).
2. Откройте Командную строку (`Win + R` → `cmd` → Enter).
3. Введите команду:
   ```cmd
   tailscale funnel --bg 8765
   ```
4. Через 5-10 секунд проверьте статус:
   ```cmd
   tailscale serve status
   ```
   Вы увидите вашу **персональную защищённую ссылку**:
   ```
   https://vasya-pc.tailxxxxx.ts.net/ (Funnel on)
   ```
   Это адрес вашего агента в интернете. Никто другой его не знает.

> 🔒 **Безопасность:** Tailscale-ссылка защищена TLS (как у банка). Без токена зайти невозможно.

### Шаг Б — передать данные ИИ

1. Откройте чат с ИИ (Arena.ai, ChatGPT, Claude или любой другой).
2. Напишите примерно так:
   ```
   У меня есть Arena Local Agent. Вот мои данные для подключения:
   URL: https://vasya-pc.tailxxxxx.ts.net/
   Token: dB9x...A2q
   Платформа: Windows
   ```
3. ИИ подключится и сможет выполнять команды на вашем ПК.

### 🔔 Звуковые уведомления
Вы можете воспроизводить звуки на компьютере прямо через Агента! Полезно запускать в конце длинных задач:
```cmd
agentctl sys beep
agentctl sys beep --type warning
agentctl sys beep --type error
```


## 6. Как пользоваться дашбордом (центр управления)

Даже без ИИ вы можете управлять агентом через красивый сайт-дашборд.

1. Откройте браузер и введите:
   ```
   http://127.0.0.1:8765/gui
   ```
2. Вы увидите:
   - **Статус служб** — зелёные значки = всё работает.
   - **Консоль** — вводите любые команды, как в терминале.
   - **Файловый менеджер** — просматривайте файлы и скачивайте отчёты.
   - **Скриншоты** — делайте снимки экрана одной кнопкой.
   - **Quick Commands** — быстрые кнопки для проверки (`/fix`, `/info`, `/svc`).

## 7. Установка плагинов MCP (расширение возможностей)

Агент поддерживает плагины MCP — это дополнительные "руки" для ИИ.

**Примеры популярных плагинов:**
- `desktop-commander` — управление рабочим столом мышью и клавиатурой.
- `filesystem` — расширенная работа с файлами.
- `fetch` — вытягивание данных из интернета.

**Как установить:**
1. Откройте PowerShell.
2. Введите:
   ```powershell
   agentctl mcp install desktop-commander
   ```
3. Агент сам скачает плагин, зарегистрирует его и скажет, как перезапустить службы.
3. Перезапустите мост:
   ```powershell
   schtasks /End /tn ArenaUnifiedBridge
   schtasks /Run /tn ArenaUnifiedBridge
   ```

## 8. Обновление Агента (1 клик)

Когда выйдет новая версия, вы получите архив `arena_agent_release.zip`.

1. Закройте дашборд и все окна агента (службы в фоне останутся работать).
2. Распакуйте новый архив **поверх** старых папок (`arena-agent` и `arena-local-bridge`) с заменой файлов.
3. Откройте папку `arena-agent` и дважды кликните:
   > 🔄 **update.bat**
4. Чёрное окно остановит старые службы, обновит файлы и запустит всё заново.
5. Готово! Ваш токен и история сохранятся.

## 9. Проверка здоровья системы

Если что-то работает не так, запустите диагностику:

1. Откройте PowerShell.
2. Введите:
   ```powershell
   agentctl sys status
   ```
   — покажет, какие порты открыты и запущены ли службы.
3. Или глубокую проверку:
   ```powershell
   agentctl sys doctor
   ```
   — автоматически исправит права на папки и проверит синтаксис.

## 10. Удаление Агента

Если нужно полностью удалить:

1. Откройте папку `arena-agent`.
2. Зажмите `Shift` и кликните правой кнопкой мыши по пустому месту → **Открыть окно PowerShell здесь**.
3. Введите:
   ```powershell
   powershell -ExecutionPolicy Bypass -File scripts\install_windows_service.ps1 -Uninstall
   ```
4. Службы удалены. Теперь можно просто удалить папки `arena-agent` и `arena-local-bridge`.

## 11. Устранение неполадок

### "Python не найден"
- Перезагрузите компьютер после установки Python.
- Убедитесь, что при установке Python стояла галочка **Add to PATH**.

### "Не могу залогиниться в Tailscale"
- Попробуйте включить VPN или Zapret.
- Или используйте альтернативу: зарегистрируйтесь на **ngrok.com**, скачайте `ngrok.exe`, запустите `ngrok http 8765` и дайте ИИ HTTPS-ссылку оттуда.

### "Порт 8765 занят"
- Перезагрузите компьютер.
- Или найдите процесс: `netstat -an | findstr 8765` и завершите его через Диспетчер задач.

### "Дашборд не открывается"
- Убедитесь, что `install.bat` успешно доработал до конца.
- Откройте Task Scheduler (`Win + R` → `taskschd.msc`) и вручную запустите задачу `ArenaUnifiedBridge`.

## 12. Безопасность и приватность

- **Токен** хранится только на вашем ПК (`token.txt`). Не выкладывайте его в публичные чаты.
- **Tailscale-ссылка** уникальна и защищена TLS. Даже если кто-то узнает ссылку, без токена зайти нельзя.
- **Агент не отправляет данные на сторонние серверы.** Всё работает локально.
- **Токен автоматически регенерируется** при каждом запуске установщика (`install.bat`). Для ручной смены токена используйте `regenerate_token.bat` в папке `arena-local-bridge`.

## 13. Краткая шпаргалка команд

| Команда | Что делает |
|---------|------------|
| `agentctl sys status` | Показать статус всех служб |
| `agentctl sys doctor` | Автоисправление ошибок |
| `agentctl do "команда"` | Поставить задачу в очередь |
| `agentctl mcp install имя` | Установить плагин MCP |
| `tailscale funnel --bg 8765` | Включить доступ из интернета |
| `tailscale serve status` | Узнать вашу публичную ссылку |
| `type %USERPROFILE%\arena-local-bridge\token.txt` | Показать токен |
| `regenerate_token.bat` | Сгенерировать новый токен и перезапустить мост |
| `update.bat` | Быстрое обновление без полной переустановки |
