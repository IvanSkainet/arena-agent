=== РУКОВОДСТВО ПО УСТАНОВКЕ И НАСТРОЙКЕ WINGET НА WINDOWS ===

Если на вашем компьютере не установлена утилита Winget (Windows Package Manager), вы можете легко и быстро установить или восстановить её следующими способами:

СПОСОБ 1: Установка через Microsoft Store (Рекомендуемый, Самый простой)
1. Откройте приложение "Microsoft Store" на вашем ПК.
2. В строке поиска введите на русском языке: "Установщик приложений" (в английской версии — "App Installer").
3. Нажмите кнопку "Получить" или "Обновить".
4. Готово! Утилита Winget установится автоматически вместе со всеми необходимыми платформенными зависимостями.

СПОСОБ 2: Установка одной командой через PowerShell
Если Microsoft Store недоступен, откройте PowerShell от имени Администратора и запустите команду:
Add-AppxPackage -RegisterByFamilyName -MainPackage Microsoft.DesktopAppInstaller_8wekyb3d8bbwe

СПОСОБ 3: Скачивание официального установщика с GitHub
1. Перейдите на страницу релизов Winget: https://github.com/microsoft/winget-cli/releases
2. Скачайте файл с расширением .msixbundle (например, Microsoft.DesktopAppInstaller_8wekyb3d8bbwe.msixbundle).
3. Дважды кликните по скачанному файлу и нажмите "Установить/Обновить".
